# Synapse Cloudflare Workers Example

Example showing how to use [@pyrx/synapse](https://www.npmjs.com/package/@pyrx/synapse) with Cloudflare Workers.

## Setup

1. `npm install`
2. `wrangler secret put SYNAPSE_API_KEY`
3. `wrangler secret put SYNAPSE_WORKSPACE_ID`
4. `npm run dev` (local) or `npm run deploy` (production)
